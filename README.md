# ROS 2 pixi example

## Installation

```bash
pixi run setup-colcon
```

## Usage

```bash
pixi shell -e jazzy
pixi shell -e humble
```

## ament_cmake_pkg

This is an example package generate from `ros2 pkg` cli

```bash
ros2 pkg create --build-type ament_cmake ament_cmake_pkg
```

## ament_python_pkg

This is an example package generate from `ros2 pkg` cli

```bash
ros2 pkg create --build-type ament_python ament_python_pkg
```
